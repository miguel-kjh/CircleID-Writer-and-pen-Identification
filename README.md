# CircleID-Writer-and-pen-Identification

Repository for CircleID Writer and Pen Identification Challenge on Kaggle. This repository contains code, notebooks, and resources related to the challenge, including data preprocessing, model training, and evaluation.

## Challenge Overview
The CircleID Writer and Pen Identification Challenge aims to develop models that can accurately identify writers and pens based on handwriting samples. Participants are provided with a dataset of handwritten samples, and the goal is to create models that can classify the writer and pen used for each sample.
